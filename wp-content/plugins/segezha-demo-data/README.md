# Segezha Demo Data Generator

Плагин для генерации демо-данных для портала Segezha.

## Возможности

### Генерация пользователей (20 пользователей)
- Имена и фамилии
- Email адреса
- Название организации (случайная из списка)
- Название подразделения (случайное из списка)
- Должность (случайная из списка)
- Дата рождения (случайная, возраст 25-55 лет)
- Множественные организации (1-2 организации на пользователя)
- Множественные подразделения (1-2 подразделения на пользователя)
- Настройки фильтрации (копируются из профиля или остаются пустыми)
- Флаг "показывать день рождения" (80% показывают)
- 1C ID (демо-значения)
- Руководители (для части пользователей)

### Генерация новостей (30 новостей)
- Заголовки новостей
- Содержание с текстом
- Авторы (случайные из созданных пользователей)
- Даты публикации (последние 3 месяца)
- Теги организаций (1-3 тега на новость)

## Использование

1. Активируйте плагин в админ-панели WordPress
2. Перейдите в **Инструменты → Демо-данные Segezha**
3. Выберите тип данных для генерации:
   - Только пользователи
   - Только новости
   - Все данные
4. Нажмите "Сгенерировать демо-данные"

Плагин создает теги для следующих организаций:

- ООО «Интеграл»
- ООО «Сегежа Пак»
- ООО «Сегежа Лес»

## Пароли пользователей

Все созданные пользователи имеют пароль: `demo123456`

## Примечания

- Плагин проверяет существование пользователей перед созданием
- Новости создаются с типом `news` (custom post type)
- Теги организаций создаются автоматически, если их нет
- Для работы требуется активированный плагин `segezha-user-sync`
